Linux常用命令或工具
2021年05月15日 614 字 大概 2 分钟

本文并非为Linux命令大全，而是为了记录下我工作中应用到的实际场景，亦是我的工作随笔(っ•̀ω•́)っ✎⁾⁾

1 查找某后缀文件并提取到指定目录

find ./ -name *.jar | xargs -i cp {} ./temp

上述命令意思是先搜索当前目录下符合筛选条件的文件，如命令中是后缀为jar的文件，然后将符合条件的全部文件都复制到当前的temp目录下，根据实际需要调整对应参数即可

2 Shell脚本部分注意事项

2.1 相对路径

在并不确定命令当前的路径是什么的时候，我们不能想当然地用./来代表当前路径

这是因为可能脚本里已经过多次的cd跳转，或者因其他命令导致路径已改变，所以“./”命令就有可能并不代表脚本所在路径。为了避免这种情况，我个人建议比较重要的脚本还得写绝对路径，要能很精准地定位，而且最好有可移植性，这样换个环境也能适用

如果一定要使用相对路径，建议在调试脚本时用pwd或者echo的方式逐步检查，否则容易酿成大祸(σﾟ∀ﾟ)σ

危险示范

rm -rf ./*
cp -r ./A ./B

建议方式

rm -rf /home/user/rightdir/*
cp -r /home/user/rightdir/A /home/user/rightdir/B

尤其是rm -rf这种危险命令，更不能在脚本中轻易使用相对路径，一不小心就会删错重要文件从而后悔莫及

常见的还有cp、tar这类可能会带有路径的命令，使用相对路径容易导致命令的执行结果不准确，且并不一定会报错，除非命令的报错条件已经设置得很完善了，我们才能够及时地发现问题

3 AIX的常见命令

[ AIX偷偷乱入一下(｡･ω･｡) ]

AIX也是工作中用过的系统，虽然与Linux看起来好像一样，但实际使用起来还是有挺多差异的地方

3.1 tar与gunzip

压缩文件成tar.gz

tar cvf - filename | gzip > finename.tar.gz

解压tar.gz文件

gunzip -c filename.tar.gz | tar -xvf -

3.2 查看进程数

lsattr -E -l sys0 | grep maxuproc

若要修改允许的最大进程数，则需要先切换到root用户，然后输入以下命令

chdev -l sys0 -a maxuproc=500

批量添加文件名后缀
2021年05月14日 358 字 大概 1 分钟

应用场景

平时处理Word文档或一些文件时，可能有这样一种需求，即需要对某些指定文件的文件名按格式批量重命名，本文主指批量的后缀添加，其他方式大同小异，此处仅提供一种方式的参考

当然，用编程的方式会做得更好，不论什么重命名方式都可以实现，但为了贪图方便，直接在工作电脑跑bat批处理文件就行~

实现代码

（1）需求：doc文档需要统一命名为Rev_3.0.doc后缀

@echo off
setlocal enabledelayedexpansion
set "rev=Rev_3.0.doc" 

for /F %%i in ('dir *.doc /b') do (
    set "var=%%~ni"
    set "var=!var!%rev%"
    ren "%%i" !var!
    echo !var! success
)
pause

（2）需求：docx文档需要统一命名为Rev_3.0.docx后缀

@echo off
setlocal enabledelayedexpansion
set "rev=Rev_3.0.docx"

for /F %%i in ('dir *.docx /b') do (
    set "var=%%~ni"
    set "var=!var!%rev%"
    ren "%%i" !var!
    echo !var! success
)
pause

食用方法

  1. 根据需要，修改 - set “rev=文件后缀”，“文件后缀”改为具体所需后缀名
  2. 将代码保存为bat文件，然后将bat文件放在文档目录下，双击运行即可

PS：此种方式的可扩展性比较低，对于不同后缀的文档，如doc、docx需要区分开，且需要 取消 在“文件夹选项”中“隐藏已知文件类型的扩展名”的勾选，但优点是很轻便而且比较无脑

多网卡路由配置
2021年05月14日 415 字 大概 2 分钟

应用场景

本地电脑有双网卡或多网卡，可通过配置对应网卡的路由网关进行网络分路

常用命令

Windows运行cmd中输入以下命令，可以检查当前路由表配置情况

route print

如需重置、初始化网络环境，可用

netsh winsock reset

配置脚本

新建一个*.bat文件，如routetest.bat，然后进行如下编辑

route delete 0.0.0.0
route add 186.1.0.0 mask 255.255.0.0 A.A.A.1
route add 10.1.0.0 mask 255.255.0.0 A.A.A.1
route add 0.0.0.0 mask 0.0.0.0 B.B.B.1
  1. A.A.A.1是A网卡的网关地址，B.B.B.1是B网卡的网关地址，如还有更多的网卡则以此类推
  2. “add”后接的IP指所需访问的网段或IP，如果子网掩码是255.255.255.255，则可指定具体IP
  3. 例，第二行命令的意思是如果访问186.1.0.0网段，那么将从A.A.A.1的网关出去，即走A网卡

网卡的对应网关，可通过cmd如下命令查询，或者进入“打开网络和共享中心”（控制面板\网络和 Internet\网络连接），右键对应网卡，选择“状态”-“详细信息”，亦可查询到默认网关

ipconfig

PS：此处有个坑，如果文件名为route.bat，那么跑bat文件就会一直执行无法停止，因此不能用该文件名
